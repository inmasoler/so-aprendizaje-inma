# DNS y dominios

<CraftCallout variant="info" title="El DNS en una frase">
  **El DNS es la agenda de teléfonos de internet.** Traduce nombres de dominio legibles (`miweb.com`) a direcciones IP numéricas. Sin entender DNS, no puedes controlar dónde apunta tu web ni tu email.
</CraftCallout>

## Flujo: ¿qué pasa cuando alguien escribe tu dominio?

1. **Browser** — escribe `miweb.com`
2. **Resolver DNS local** — ¿lo tengo en caché?
3. **Root DNS** — ¿quién gestiona `.com`?
4. **Nameserver del dominio** — tabla de registros DNS
5. **IP del servidor** — registro A
