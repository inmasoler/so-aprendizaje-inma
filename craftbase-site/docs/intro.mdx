## Ejemplo de código en vivo

Los bloques `live` permiten editar y ejecutar ejemplos (útil para JavaScript y snippets interactivos):

```jsx live
function Saludo() {
  const [n, setN] = React.useState(0);
  return (
    <button type="button" onClick={() => setN(n + 1)}>
      Hola CraftBase ({n})
    </button>
  );
}
```

Para comandos de terminal, Docker o SQL, usa bloques estáticos con resaltado de sintaxis.

```bash
npm run dev:v1
```
